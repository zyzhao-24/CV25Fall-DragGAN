# Area-wise Tracking Method Based on Texture Variance

## 方法评价

针对点跟踪任务，混合使用 RAFT 和 L2 跟踪方法可以在不同纹理区域获得更优效果。具体来说：
- 在无纹理（Textureless）区域，RAFT 方法表现更好，能够更准确地跟踪点。
- 在高纹理（Texture rich）区域，L2 跟踪方法更为精确。

混合方案虽然在高纹理区域效果理想，但在无纹理区域仍存在一定偏差。为此，提出了一种基于区域方差（texture variance）的 area-wise 跟踪方法：

1. **区域方差判定纹理丰富度**：通过计算点邻域的图像方差，自动判断该区域属于高纹理还是低纹理。
2. **自适应选择跟踪方法**：
   - 若方差较高，判定为高纹理区域，采用 L2 跟踪。
   - 若方差较低，判定为无纹理区域，采用 RAFT 跟踪。

### 优点
- **自适应性强**：根据实际区域特性动态选择最优跟踪方法，提升整体跟踪精度。
- **鲁棒性提升**：有效规避单一方法在特殊区域的失效问题。
- **理论合理**：纹理方差是衡量区域特性的有效指标，能够反映跟踪难度。

### 潜在问题
- **方差阈值选择**：如何设定高低纹理的分界阈值需要实验验证。
- **计算开销**：每步都需计算区域方差，可能增加一定计算量。
- **边界情况**：在纹理过渡区域，方法选择可能不够稳定。

### 总结
基于区域方差的 area-wise 跟踪方法是一种合理且有效的改进方案，能够针对不同区域特性自适应选择 RAFT 或 L2 跟踪，提升整体点跟踪的准确性和鲁棒性。
